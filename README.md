# 🎬 Streaming Backend API

Backend service for the Streaming Web App.
Built with Node.js, Express, MongoDB.

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Docker
- AWS EC2
- Caddy (Reverse Proxy)
